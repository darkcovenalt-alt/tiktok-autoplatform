Política de Privacidade

Última atualização: 29 de junho de 2026

1. Introdução

Esta Política de Privacidade descreve como o TikTok AutoPlatform coleta, usa e protege suas informações.

2. Dados Coletados

Coletamos apenas os dados necessários para a operação do serviço:

  • Informações de perfil público do TikTok (username, avatar)
  • Tokens de acesso OAuth (armazenados de forma segura)
  • Conteúdo de vídeo enviado para publicação
  • Métricas de desempenho das publicações (visualizações, engajamento)

3. Uso dos Dados

Os dados são utilizados exclusivamente para:

  • Publicar conteúdo automático no TikTok
  • Analisar desempenho das publicações
  • Melhorar o serviço de recomendação de conteúdo

4. Armazenamento e Segurança

Os dados são armazenados em servidores próprios com criptografia. Tokens de acesso são renovados periodicamente e armazenados com segurança em banco de dados PostgreSQL.

5. Compartilhamento

Não compartilhamos dados com terceiros. O conteúdo publicado no TikTok segue as políticas de privacidade da própria plataforma.

6. Direitos do Usuário

Você pode revogar o acesso do nosso aplicativo a qualquer momento através das configurações do TikTok. Para solicitar remoção de dados, entre em contato.
